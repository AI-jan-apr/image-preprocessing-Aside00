# README

## Task Goal

The goal of this task is to practice **basic image processing using OpenCV**. The task focuses on understanding how to manipulate colors in an image and how to work with masks.

## What I Did

1. **Color Detection**

   * I created a mask to detect the **red color** in the image.

2. **Grayscale Conversion**

   * The original image was converted to **grayscale** so that all colors become gray.

3. **Masking**

   * The red color area was kept in its original color.
   * Everything else in the image was converted to grayscale.

4. **Combining Results**

   * Finally, the red colored part and the grayscale background were combined to produce the final image.

## Result

The final result shows:

* **The red umbrella in color**
* **All other parts of the image in grayscale**
